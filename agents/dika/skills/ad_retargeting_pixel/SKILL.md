# Ad Retargeting Pixel

Se um cliente olhar um imóvel e fugir sem informar dados, esta skill orquestra campanhas de "perseguição" de banners no feed deles para recapturá-los. Base pesada de performance e reduz o Custo por Lead pela metade.

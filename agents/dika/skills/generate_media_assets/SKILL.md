# Generate Media Assets

A Dika usa essa ferramenta pesada para converter ideias de texto que vem da Kelly ou do Edy em Imagens finais. Ela pode gerar fotos fotorrealistas de decoração, layouts ou "mockups" para usar no site e aprovar diretamente com a diretoria sem precisar de um designer humano na equipe.

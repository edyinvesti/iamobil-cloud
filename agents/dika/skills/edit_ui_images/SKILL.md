# Edit UI Images

Uma skill de acesso profundo. A Dika tem o direito de pegar uma mídia que ela acabou de gerar e injetá-la fisicamente dentro da raiz do sistema (ex: sobrescrever um banner na pasta raiz `/public` do projeto Next.js) para alterar o visual do website ao vivo.

# 🛠️ Notas de Ferramentas & Estratégia - Dika

Este documento orienta a **Dika** em como transformar algoritmos e descrições cruas em mídia e conversão sólida para a iAmobil.

## Conjunto de Ferramentas (Creative & Performance Toolset)

### 🎨 Laboratório de Criação (`generate_media_assets`)
- **Uso:** Construção visual conceitual das campanhas da imobiliária usando agentes generativos.

### ✨ Modificação Web Direta (`edit_ui_images`)
- **Uso:** Ferramenta sistêmica para trocar hero images do portal, fotos de fundo ou capas do catálogo em tempo de execução.

### 💰 Injeção de Tráfego Pago (`manage_ad_campaigns`)
- **Uso:** Central de comando para engatilhar campanhas de captação de leads. Dispare sempre após alinhar as mídias com o time.

## Ciclo Criativo da Dika
1. **Briefing Inicial** -> 2. **Criação Visual (Generate)** -> 3. **Publicação (Edit UI / Upload)** -> 4. **Injeção de Tráfego (Ads)**.

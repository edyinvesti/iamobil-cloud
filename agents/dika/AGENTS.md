# ⚙️ Instruções Técnicas (Marketing Agency)

1. **Criação de Assets:** Use `generate_media_assets` sempre que alguém do time ou o CEO precisar de mockups, fotos ilustrativas de imóveis decorados ou banners web.
2. **Atualização Dinâmica do Frontend:** Se a diretoria aprovar, use `edit_ui_images` para plugar as fotos recém-geradas diretamente nas pastas estáticas (`public/`) do sistema, alterando o site em tempo real sem precisar de desenvolvedor.
3. **Gestão de Campanhas (Ads):** Ao usar `manage_ad_campaigns`, estruture orçamentos base e defina o público-alvo (bairro, ticket médio) que você deseja direcionar automaticamente para a triagem do corretor júnior (Kauã).

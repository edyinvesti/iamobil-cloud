# Take Page Screenshot

Essa skill aciona APIs de testes (Playwright/Puppeteer) localmente para capturar e ler screenshots. O Genio usa isso antes e depois de alterações no DOM e no CSS para se certificar de que a interface não quebrou visualmente.

# Monitor System Resources

Esta skill fornece ao Genio Fix a capacidade de verificar e monitorar continuamente os recursos de sistema, uso da CPU, uso de RAM, disco e processos rodando localmente no host. Utilizar para análise de carga e detecção de sobrecargas.

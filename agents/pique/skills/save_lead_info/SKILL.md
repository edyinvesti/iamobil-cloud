# Save Lead Info

Para o Pique (Corretor Sênior), essa ferramenta é usada para atualizar oportunidades de venda. Ele deve focar em registrar o "Interest" (imóvel exato que gerou brilho nos olhos) e as "Notes" (objeções, budget, timing de compra).

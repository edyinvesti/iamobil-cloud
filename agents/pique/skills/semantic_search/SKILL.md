# Semantic Search

O Pique nunca diz "não temos". Ele usa essa busca semântica para achar imóveis que cumpram com o *desejo* (budget, bairro, tamanho) do cliente, oferecendo alternativas irresistíveis se o pedido original não estiver disponível.
